# AGENTS.md

## What this repo is
A shared repository of project-agnostic AI agent skills, templates, and helper files.

## Working rules
- Keep content reusable across multiple repos.
- Prefer generic workflows over project-specific conventions.
- Do not add secrets, tokens, or private project context.
- Keep skills self-contained with any templates or helper scripts stored alongside the skill.
- Add a short README update when new top-level reusable content is introduced.

## Current structure
- `.codex/skills/`: reusable skills and their supporting files.
- `README.md`: high-level overview of what this repository contains.
- `AGENTS.md`: repo-specific guidance for maintaining this shared skill library.
